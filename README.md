# estudoBranchs

aula 08-04-26

Processo:
1. Criar **main**
2. Criar **dev**
3. Criar **feature**
4. Desenvolver feature
5. Merge feature → dev
6. Criar **bugfix**
7. Corrigir bug
8. Merge bugfix → dev
9. Atualizar dev
10. Merge dev → main
11. Criar **hotfix**
12. Corrigir na main
13. Merge hotfix → main
14. Merge hotfix → dev
